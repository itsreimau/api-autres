AutresGPT Web Server Usage Guide with AutoResponder

1. Adding a New Rule in AutoResponder

- Open the AutoResponder application on your Android device.
- Select the option to add a new rule.

2. Configuring Rules in AutoResponder

- In your new rule, specify the received messages that should be processed by the AutresGPT Web Server.
- It is recommended to use the "All" button in the upper right corner to process all messages with your web server.

3. Setting up Web Server Options in AutoResponder

- Choose the "Web Server" option in your AutoResponder rule.
- Enter the complete URL of the AutresGPT web server provided by your provider (example: https://x-arai.bohr.io/api/index.php).

4. Configuring the "commands" Header (Optional)

- Add the "commands" header to process messages with specific commands.
- Example: commands: ^/chatgpt\s* (will capture the part of the message after the "/chatgpt" command).

5. Additional Options (Optional)

- Basic Auth and Headers options are optional. For more information, refer to the comments in the PHP code above if you wish to use them. It is recommended to add authorization if your API provides access to sensitive information.

6. Finished!

- You are now ready to test your web server integration.
- Send messages and check if the AutresGPT Web Server provides the expected responses.

1. Menambahkan Aturan Baru di AutoResponder

- Buka aplikasi AutoResponder di perangkat Android Anda.
- Pilih opsi untuk menambahkan aturan baru.

2. Konfigurasi Aturan di AutoResponder

- Di aturan baru Anda, tentukan pesan yang diterima yang harus diproses oleh AutresGPT Web Server.
- Disarankan menggunakan tombol "All" di sudut kanan atas untuk memproses semua pesan dengan web server Anda.

3. Mengatur Opsi Web Server di AutoResponder

- Pilih opsi "Web Server" di aturan AutoResponder Anda.
- Masukkan URL lengkap dari web server AutresGPT yang diberikan oleh penyedia (contoh: https://x-arai.bohr.io/api/index.php).

4. Mengatur Header "commands" (Opsional)

- Tambahkan header "commands" untuk memproses pesan dengan perintah khusus.
- Contoh: commands: ^/chatgpt\s* (akan mengambil bagian pesan setelah perintah "/chatgpt").

5. Opsi Tambahan (Opsional)

- Opsi Basic Auth dan Headers bersifat opsional. Untuk informasi lebih lanjut, lihat komentar dalam kode PHP di atas jika Anda ingin menggunakannya. Disarankan untuk menambahkan otorisasi jika API Anda memberikan akses ke informasi sensitif.

6. Selesai!

- Anda sekarang siap untuk menguji integrasi web server Anda.
- Kirimkan pesan dan periksa apakah AutresGPT Web Server memberikan respons yang diharapkan.
